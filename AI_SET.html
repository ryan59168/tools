💡 如何在電腦上新增/刪除工具？

這個頁面完全是靜態且免資料庫的！您只需要在電腦上用任何文字編輯器（如記事本、VS Code）開啟本 ai-links.html 檔案，尋找頂部的 AI_TOOLS 陣列，直接修改該 JavaScript 物件即可，無需編寫任何複雜程式碼。

➕ 新增 AI 工具按鈕
1
用文字編輯器開啟檔案，在最前方找到 const AI_TOOLS = [ 一行。
2
在陣列結尾的 } 後方，補上一個逗號 ,，並貼上以下格式：
{ name: "工具名稱", url: "https://工具網址", desc: "一行簡短的功能介紹說明", category: "分類名稱", color: "#4F46E5" // 卡片流光自訂顏色 }
3
替換引號內文字為您想要的工具資訊，存檔後在瀏覽器中重新整理即可生效。
🗑️ 刪除或調整順序
①
刪除： 尋找特定的 { ... } 物件，將其完整選取並刪除即可。
②
排序： 透過「剪下/貼上」整組 { ... } 物件，即可直接調整工具在網頁上的排列順序。
🏷️ 分類設定說明

系統會自動偵測 category 欄位，只要您填入了全新名稱（例如 "翻譯"），篩選列將自動顯示該標籤，無需手動修改 HTML 排版。
